# DeepSeek Balance

macOS 菜单栏余额查询工具 — 实时查看你的 DeepSeek API 余额。

## 功能

- 🔍 实时查询 DeepSeek API 余额
- 🖥️ 菜单栏显示，不打扰工作
- 🔑 支持随时修改 API Key
- 🔄 每 5 分钟自动刷新
- 🚀 开机自启

## 安装

```bash
git clone https://github.com/YOUR_USERNAME/DeepSeekBalance.git
cd DeepSeekBalance
chmod +x install.sh
./install.sh
```

或者直接下载 `DeepSeekBalance.app` 拖入「应用程序」文件夹。

## 使用

1. 打开应用
2. 输入你的 DeepSeek API Key（在 [platform.deepseek.com](https://platform.deepseek.com) 获取）
3. 右上角菜单栏就会显示余额
